# To-Do-List

A simple and interactive To-Do Board built with HTML, CSS, and JavaScript. This project allows you to organize tasks across multiple lists, mark tasks as completed, and reorder items using drag-and-drop. All data is saved in the browser using Local Storage, so your tasks remain even after closing the page (as long as the cache is not cleared).

Features

-Create multiple lists to organize tasks.
-Add, edit, and delete tasks easily.
-Mark tasks as completed.
-Drag and Drop support to reorder tasks within a list.
-Persistent data storage using Local Storage.
